# Secure Explainable AI Financial Advisor Bot

This project implements a secure, explainable AI-based financial advisor using
reinforcement learning, surrogate explainability models, and a constrained
language interface.

## Features
- Reinforcement learning trading agent
- Explainable AI using SHAP
- Secure LLM explanation pipeline
- Simulated user portfolios
- Web-based user interface

## Disclaimer
This system is for educational and research purposes only and does not provide
financial advice.
